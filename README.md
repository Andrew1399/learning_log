# learning_log
Application for creating topics and entries


![image](https://user-images.githubusercontent.com/102066744/224691559-a0ec4907-95f6-4c6d-81b0-51e854078446.png)

# Technology Stack
- Python 3.9
- Django 4.1
- Django allauth
- Sqlite3
- Bootstrap4
